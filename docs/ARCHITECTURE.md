# 博客系统架构设计

## 📐 系统架构图

```
┌─────────────┐     ┌─────────────┐
│   Client    │────▶│   Nginx     │
└─────────────┘     └──────┬──────┘
                           │
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
    ┌────────────┐   ┌────────────┐   ┌───────────┐
    │  Vue3    │   │  Spring  │   │   H2     │
    │  Frontend│   │  Boot    │   │ Database  │
    └────────────┘   └────────────┘   └───────────┘
                           │
                    ┌──────┴──────┐
                    │ RAG Service │
                    │ (TF-IDF)    │
                    └─────────────┘
```

## 🏗️ 技术栈

| 层级 | 技术 |
|------|------|
| Frontend | Vue3 + Vite + Axios |
| Backend | Spring Boot 3 + JPA |
| Database | H2 (内存模式) |
| RAG | Java TF-IDF |
| Web | Nginx |
