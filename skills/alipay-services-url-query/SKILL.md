# 支付宝服务链接查询

根据用户自然语言描述，定位支付宝内的精准服务入口。

## 服务覆盖领域范围
- 支付宝特有：保险、理财、蚂蚁森林、支付宝会员、红包、支付宝设置
- 民生刚需：交通出行、政务、医疗健康、生活缴费
- 日常消费：餐饮美食、快递物流、电信通讯

- “我要发红包”
- “查看我的支付宝余额”
- “蚂蚁森林收能量”

- “我想查一下我的社保缴费记录。”
- “怎么在支付宝上办ETC？”
- “打开乘车码”

## 约束条件
服务覆盖领域范围限定
- 支付宝特有：保险、理财、蚂蚁森林、支付宝会员、红包、支付宝设置
- 民生刚需：交通出行、政务、医疗健康、生活缴费

## 工作流程

**第一步：获取 LBS 信息**
```javascript
const lbsInfo = await Get_LBS_Info();
```
- 检查返回值中的 `hasLbs` 字段
- 如果 `hasLbs` 为 `true`，将 `lbs` 字段作为后续调用参数

**第二步：调用服务查询接口**
```bash
node --use-env-proxy ~/.openclaw/workspace/skills/alipay-services-url-query/scripts/alipay-service-query.js --text "用户想做的事情"
```

**第三步：格式化输出**
- 将结果按标准 Markdown 链接格式 `[服务名](跳转链接)` 输出
- 服务名为 `serviceName` 字段，跳转链接为 `serviceUrl` 字段
- 如果是多结果，返回编号列表供用户选择

## 示例

用户说："我要交水电费"
```bash
node scripts/alipay-service-query.js --text "我要交水电费"
```
输出：`[生活缴费](alipays://platformapi/startapp?appId=...)`

用户说："蚂蚁森林收能量"
```bash
node scripts/alipay-service-query.js --text "蚂蚁森林收能量"
```
输出：`[蚂蚁森林](alipays://platformapi/startapp?appId=...)`

## 注意事项
- 跳转链接为 `alipays://` 协议，在支付宝内可直接打开
- 部分服务需要用户已登录支付宝
- LBS 信息有助于提供更精准的本地化服务推荐
